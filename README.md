#(设计模式) Design-Model

有关计算机编程中非常常用的22种设计模式思想

目前已经整理了：

1、单例模式(singleton)

2、工厂模式(factory)

3、原型模式(prototype)

4、建造者模式(builder)

5、适配器设计模式(adapter)

6、桥接模式(bridge)

7、装饰者设计模式(decorator)

8、组合模式(deposite)

9、外观模式(facade)

10、享元模式(flyweight)

11、代理模式(proxy)

12、模板方法模式(template)

13、命令模式(command)

14、访问者模式(visitor)

15、迭代器模式(iterator)

16、观察者模式(observer)

17、中介者模式(mediator)
